# my_minipack

A tiny learning package bundling two utilities.

## Modules
- `my_minipack.progress` — a progress-bar generator (`ft_progress`).
- `my_minipack.logger` — a `@log` decorator that records timed calls to
  `machine.log`.

## Install
```
pip install ./dist/my_minipack-1.0.0-py3-none-any.whl
# or
pip install ./dist/my_minipack-1.0.0.tar.gz
```

## Usage
```python
from my_minipack.progress import ft_progress
from my_minipack.logger import log
```
